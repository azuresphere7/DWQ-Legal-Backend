# DWQ Legal ( Backend )

## Technologies
- Node.js
- Express.js
- Typescript
- Json Web Token
- Passport
- AWS Cognito
- AWS DynamoDB

<br />

## Available Scripts

### `pnpm lint`

Runs ESLint Configuration.

### `pnpm server:start`

Runs the server for the product. \
Call [http://localhost:8000/api](http://localhost:8000/api) to fetch APIs from the server.
